# Lotto Number Recommendation Site

## Overview

This project is a simple web application that generates and displays random lottery numbers. It's built with HTML, CSS, and JavaScript, following modern web development practices.

## Features

*   **Lotto Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Modern UI:** A clean and visually appealing user interface.
*   **Responsive Design:** The layout adapts to different screen sizes.
